## lancer un projet : 


installer .net6

https://dotnet.microsoft.com/en-us/download

sinon:

```
./dotnet-install.sh --channel 6.0 --runtime aspnetcore     

```


# verifier avec 

```
dotnet --version  

```


```
dotnet new blazorwasm -f net6.0 -o Nom projet

```   

puis lancer le déboguage
